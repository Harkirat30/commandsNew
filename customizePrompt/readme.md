
Common special characters 

1) \d - the current date
2) \e - escape character
3) \h - hostname
4) \n - new line
5) \t - current time 24-hour HH:MM:SS
6) \T - 12-hr HH:MM:SS
7) \@ - 12-hr HH:MM am/pm
8) \u - username of current user
9) \w - path to current working dir
